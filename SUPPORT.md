# Support

Use GitHub issues for community support, bug reports, and feature requests.

Before opening an issue:

- Search existing issues for similar reports.
- Confirm the behavior on the current main branch when practical.
- Remove secrets, tokens, private paths, and sensitive log content.

For bugs, include your operating system, Python version, command used, expected behavior, actual behavior, and a small reproducible example when possible.

This is a community project and does not provide a guaranteed response time. For suspected security vulnerabilities, follow `SECURITY.md` instead of posting details publicly.
